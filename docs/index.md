注：
- 当前最频繁的琐事：规划实现的细节设计。比如看什么书、做什么项目、关注哪些资讯等。
- 一个月内设计上应无大变动，多数进入常态化。

- 时间区划分：
    - 工作日工作区、工作日休息区、工作日充电区、工作日在途区
    - 非工作日工作区、非工作日休息区、非工作日充电区、非工作日在途区
